# Background
It's Thomas Bangalter's helmet from Daft Punk

It is based on an [adafruit tutorial](https://learn.adafruit.com/3d-printed-daft-punk-helmet-with-bluetooth?view=all)

Hardware:
* [Adafruit Feather 32u4 Bluefruit](https://www.adafruit.com/products/2829)
* 2x [Adafruit Mini Skinny NeoPixel Digital RGB LED Strip](https://www.adafruit.com/product/2964?length=1)

The lights are laid out in a 15 x 8 grid

I started with this code:
* [Adafruit's demo code](https://learn.adafruit.com/system/assets/assets/000/029/390/original/feather_bluefruit_neopixel_animation_controller.zip?1451271686)
* [Twilio's example using Adafruit's NeoMatrix Library](https://www.twilio.com/blog/2015/05/light-up-your-hackpack-with-the-adafruit-neomatrix-library.html)
* [Adafruit's strandtest](https://github.com/adafruit/Adafruit_NeoPixel/blob/master/examples/strandtest/strandtest.ino)
